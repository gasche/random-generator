Module Random_generator.Fuel

The Fuel submodule provides the basic operations on fueled generators.

val map : ('a -> 'b) -> 'a fueled -> 'b fueled
val map' : 'a fueled -> ('a -> 'b) -> 'b fueled
val zero : 'a -> 'a fueled
val tick : 'a fueled -> 'a fueled
val prod : (int -> (int * int) gen) -> 'a fueled -> 'b fueled -> ('a * 'b) fueled
val choose : 'a fueled list -> 'a fueled

For a given amount of fuel, we will only choose among the generators that can produce a term.
